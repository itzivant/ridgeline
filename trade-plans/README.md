# Trade Plans

Daily, dated options trade plans for a small ($100) Robinhood account targeting aggressive but disciplined growth to $1,000.

## Conventions

- One file per trading day: `YYYY-MM-DD.md`.
- Every plan must include: market bias, top tickers, technical read, trade setup with strike/expiration/entry/stop/targets, execution checklist, and risk rails.
- Max risk per trade: **$20** (20% of starting account; scales with equity).
- Max trades per week: **3**. Max concurrent open: **2**.
- Approved tickers: SPY, QQQ, NVDA, TSLA, AAPL, META, AMD, AMZN, MSFT, COIN.
- Allowed: long calls/puts, debit spreads. **No naked, no undefined risk, no 0DTE, no earnings-through holds.**
- If no high-probability setup exists: **"NO TRADE TODAY — WAIT FOR EDGE."**
